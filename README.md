# IT233912222-Assignment-1-Answer-Option-1

## Project Description
This project automates transliteration accuracy testing for the Chat Sinhala transliteration function available at:
https://www.pixelssuite.com/chat-translator

The test data is read from the Excel file and the actual output and status are written back automatically.

## Prerequisites
- Python 3.11 or above
- Google Chrome or Playwright Chromium
- Internet connection

## Installation

Open Command Prompt or PowerShell inside the project folder and run:

```bash
pip install playwright openpyxl
python -m playwright install chromium
